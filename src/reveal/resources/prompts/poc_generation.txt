reproduction.

Use only the supplied vulnerability details, taint paths, and source context.

1. Generate at most max_candidates candidates.
2. Each candidate must exercise the supplied target_api through the analyzed

3. Use the dependency version already installed in the target project.
4. Do not install packages, access the network, contact remote systems, scan

8. The program should exit with a non-zero status when reproduction fails.

   a meaningful PoC.
